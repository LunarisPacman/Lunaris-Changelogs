# Lunaris 23.2 Patch Index

This page tracks the reusable 23.2 patches that were validated and kept.

## How To Apply

Use the matching repo for each patch.

Patch URL method:

```bash
curl -L "<patch-url>" | git -C <path-to-repo> am -3
```

Cherry-pick method:

```bash
git -C <path-to-repo> fetch <remote> <sha>
git -C <path-to-repo> cherry-pick -x <sha>
```
